0. Pertama-tama anda akan di tuju pada halaman home yang terdapat logo, deksripsi tentang aplikasi dan hewan adopsi.
2. jika ingin melihat lebih banyak lagi hewan adopsinya , silahkan klik hewan pada navbar di kanan atas
3. klik "lihat aku" untuk melihat deskripsi tentang hewan.
4. jika anda mau mengadopsi, anda harus login terlebih dahulu. jika belum, register terlebih dahulu baru kemudian login kembali.
5. jika anda setuju dengan persyaratan atas donasi, maka klik ok.
6. pengadopsian hewan akan masuk kedalam daftar adopsi, silahkan tunggu kode dari admin untuk mengambil hewan tersebut.
7. setelah kode tersebut tersedia, silahkan bertemu dengan salah satu admin kita dengan membawa bukti adopsi.

------admin-------
1. Pertama-tama anda akan di tuju pada halaman home yang terdapat logo, deksripsi tentang aplikasi dan hewan adopsi.
2. ubah link nya menjadi localhost/adopsi-hewan/adopsi-hewan-pihak-admin
3. silahkan masukkan data dengan benar.
4. anda akan tertuju pada halaman data admin tentang presentase yang sudah diadopsi dan pengajuan kode untuk adopsi
5. anda bisa menambah data di file Jenis hewan dan Hewan.
6. file adopsi ialah, data hewan yang teradopsi.
7. pada file adopsi, klik jenis jenis hewan tersebut untuk membuatkan sertifikat kepada pemilik.
